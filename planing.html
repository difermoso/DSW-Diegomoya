GIT y GITHUB

Ojetivo general

Analizar la importancia de llevar un control de versiones (git), tener la facilidad de crear repositorios y mantener una copia en la nube (github).

Objetivos especificos

  • Profundizar en la importancia de poder realizar un control de versiones.
  • Realizar comparaciones entre dos versiones.
  • Poder fusionar cambios entre distintas versiones.
  • Trabajar con distintas ramas, por ejemplo la produccion y el desarrollo.
